# Mntone UI Framework

This is UI Framework for Windows like xaml.ã€€The next program is sample code.

```C++
text_box* tb = new text_box(L"initial text.");
button* btn = new button(L"push button sample");
grid* root = new grid(
{ grid_length(), 28 },
{ },
{
	make_tuple(0, 0, tb),
	make_tuple(1, 0, btn),
});

unique_ptr<window> wnd = make_unique<window>();
wnd->set_child(root);
wnd->set_margin(9);
wnd->set_text(L"guitest window");
```
